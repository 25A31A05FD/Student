# Number Puzzle Game with Scoreboard Saved to File

**A number puzzle game that tracks scores and saves them to a file.**

## Features
- Classic number puzzle gameplay (sliding numbers to arrange in order).  
- Keeps track of moves and completion time.  
- Scoreboard that saves high scores to a file for future sessions.  
- Simple and easy-to-use interface.  

## How to Play
1. Launch the game.  
2. Shuffle the numbers and try to arrange them in ascending order.  
3. Each move is counted, and your score is calculated based on performance.  
4. Check the scoreboard to see saved high scores.  

## Technologies Used
- Programming Language: (e.g., Python / Java / C++)  
- File handling for storing scoreboard data
